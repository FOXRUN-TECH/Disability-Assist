# Contributors

This file recognizes all contributors to the Disability-Assist project.
It is automatically updated by CI on each merge to the main branch.

Contributions include code, documentation, design, testing, accessibility
review, and governance work. All contributors are recognized regardless
of contribution size.

## How Attribution Works

- **Automatic**: The CI workflow extracts contributors from git history
  and updates this file on each merge to `master`.
- **Manual**: Contributors may also be added manually for non-code
  contributions (design, accessibility review, pilot feedback).
- **NOTICE file**: The [NOTICE](NOTICE) file (required by Apache 2.0)
  references this file and lists third-party dependencies.
