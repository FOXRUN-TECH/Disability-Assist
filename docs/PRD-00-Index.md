# Disability-Assist Replacement PRD Set

This package replaces the earlier draft set with a cloud-first, pilot-ready architecture and a stricter privacy/governance posture.

Included files:
- PRD-01 Overview & Vision
- PRD-02 User Personas & Journeys
- PRD-03 System Architecture
- PRD-04 Voice Input & Cloud STT
- PRD-05 Intent Engine & Action Safety
- PRD-06 Adaptive Voice Output
- PRD-07 Smart Home & Device Integration
- PRD-08 Personalization & User Memory
- PRD-09 Caregiver Communication Bridge
- PRD-10 Hardware Reference Design
- PRD-11 Privacy, Security & Governance
- PRD-12 Open Source Contribution Guide
- PRD-13 Cloud RAG & Cross-User Assistive Learning
- PRD-14 Caregiver Mobile App

Key changes:
- banned provider name removed entirely;
- cloud-first v1 made explicit;
- caregiver app moved into v1;
- assisted-home pilot scope formalized;
- “personality development” reframed as assistive personalization and cross-user priors;
- privacy, consent, and role separation tightened;
- unrealistic hardware cost target corrected.
